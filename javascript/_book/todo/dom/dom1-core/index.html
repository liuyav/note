dom1-core 描述的是层次化的节点树，允许开发人员修改、添加、移除页面的某一部分

  • 节点层次：
    1. 成树状向下分布
    2. 在html中根节点只有一个为html，在xml中任何节点都可以成为根节点
  • 节点类型：
    1. nodeType一共12种，常用类型4种（el：1、attr：2、text：3、node：8、doc：9）
      • Element类型：
        • HTMLElement类型继承于Element类型，html元素是HTMLElement类型的一个实例
        • html元素属性（id、title、lang、dir、className、attributes）
        • html元素方法（getAttribute()、setAttribute()、removeAttribute()、createElement()）
      • Document类型：
        • HTMLDocument类型继承于Document类型，document对象是HTMLDocument类型的一个实例
        • document对象属性（documentElement、body、doctype、title、url、domain、referrer）
        • document对象查找元素方法（getElementByID()、getElementsByTagName()、getElementsByName()）,返回一个HTMLCollection对象
        • document对象特殊集合（anchors、forms、images、links）
        • document对象写入方法（write()、writeIn()、open()、close()）
    2. nodeName始终为标签名
    3. nodeName始终为空
    4. 节点间层级关系
      • childNodes表示节点的子节点，nodeList对象
      • parentNode表示节点的唯一父节点
      • previousSibling/nextSibling表示节点的上一个/下一个兄弟节点
      • firstChild/lastChild表示节点的第一个/最后一个子节点
      • 所有节点都有一个属性为ownerDocument，指向整个文档的文档节点（html）
      • hasChildNodes()方法在节点树大于0时候会返回ture
  • 操作节点：
    1. appendChild()方法向一个childNodes列表末尾添加一个节点，返回新增节点
    2. insertBefore()方法向一个childNodes列表中得某一节点前添加一个节点，返回新增节点
    3. replaceChild()方法，替换节点，返回被替换的节点
    4. removeChild()方法，移除节点，返回被移除的节点
    5. cloneNode()复制节点，执行深复制时候，参数为true
    6. normalize()处理文档树中的文本节点
