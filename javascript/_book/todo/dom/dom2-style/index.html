dom2-style围绕定义样式的三种方式提供了一套API

  • style属性：
    1. style对象是CSSStyleDeclaration的实例，包含通过style 特性指定的样式，但不包括外部与内嵌样式
    2. 对于分割属性转化成驼峰形式
    3. cssText包含style中所有样式信息，可以被重写
    4. float在样式对象上对应的属性应该是cssFloat，IE下是styleFloat
    5. 属性方法：
      • cssText：访问style中的css代码
      • length：css属性数量
      • parentRule：表示CSS信息的CSSRule对象
      • getPropertyCSSValue(attr)：返回包含给定attr对应的CSSValue对象
      • getPropertyPriority(attr)：如果给定属性使用了 !important 则返回该字符，否则为空
      • getPropertyValue(attr)：返回给定attr的字符串值
      • removeProperty(attr)：从样式中删除给定属性
      • item(index)：返回给定位置css属性名称
      • setProperty(attr, value, priority)：给给定属性设置相应的值
  • link标签/style标签：
    1. link标签HTMLLinkElement类型，style标签是HTMLStyleElement类型，都表示样式表，用CSSStyleSheet类型表示更通用一些
    2. 继承自StyleSheet类型，继承属性如下：
      • disabled
      • href
      • media
      • ownNode
      • parentStyleSheet
      • title
      • type
      • cssRules表示的是样式表中的每一条规则，IE下为rules
        • cssText不同于style.cssText，不可写
        • parentRule
        • parentStyleSheet
        • selectorText
        • style：一个CSSStyleDeclaration对象
        • type
      • ownerRule
      • deleteRule(index),IE下为removeRule(index)
      • insertRule(rule, index)，IE下为addRule(slelcore, rule, index)
    3. 获取样式表集合：（styleSheets属性、取得link标签下的sheet/styleSheet属性）
  • 获取最终样式（document.defaultView.getComputedStyle()、currentStyle）
